# TrioHub — Community Skills & Plugins for trio.ai

Browse, install, and share AI skills and plugins.

## Install a skill
```bash
trio skill install <name>
```

## Search skills
```bash
trio hub search "coding"
```

## Browse categories
```bash
trio hub trending
```

## Publish your own skill
```bash
trio hub publish my-skill.md
```
