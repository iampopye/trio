# Landing Page CRO Analysis

Perform comprehensive Conversion Rate Optimization analysis on any landing page with section-by-section scoring and prioritized, actionable fixes.

## When to Use

- Auditing a landing page for conversion optimization
- Improving signup, lead capture, or purchase rates
- Preparing A/B test hypotheses
- Reviewing landing page design and copy effectiveness

## Step 1: Identify Page Type

| Page Type | Primary Goal | Good CR | Great CR |
|-----------|-------------|---------|----------|
| Lead Capture | Email/form submission | 5-10% | 15%+ |
| SaaS Signup | Free trial or freemium | 3-7% | 10%+ |
| E-commerce Product | Add to cart / Purchase | 2-4% | 5%+ |
| Webinar Registration | Register for event | 20-30% | 40%+ |
| Waitlist | Join waitlist | 15-25% | 35%+ |
| Consultation Booking | Schedule a call | 5-10% | 15%+ |

## Step 2: 7-Point CRO Framework

### Section 1: Hero Section (Weight: 25%)

**Checklist:**
- [ ] Headline visible within 2 seconds of page load
- [ ] Headline communicates primary benefit (not a feature)
- [ ] Headline under 10 words
- [ ] Subheadline adds specificity
- [ ] Primary CTA above the fold
- [ ] CTA button color contrasts with background
- [ ] CTA text is action-oriented
- [ ] Hero image/video supports the message
- [ ] Trust badges or social proof visible above fold
- [ ] Page loads under 3 seconds
- [ ] No navigation competing with CTA (for dedicated landing pages)

### Section 2: Value Proposition (Weight: 20%)

Evaluate using the 4U Framework:
1. **Useful** -- Does it solve a real problem?
2. **Urgent** -- Is there a reason to act now?
3. **Unique** -- Is it different from competitors?
4. **Ultra-specific** -- Are claims concrete?

### Section 3: Social Proof (Weight: 15%)

**Types ranked by persuasion power:**
1. Revenue/results metrics ("$2.4B processed", "500K users")
2. Named testimonials with photos, titles, companies
3. Recognizable client logos
4. Case studies with specific results
5. Star ratings and review counts
6. Media mentions
7. Certifications and awards

### Section 4: Features and Benefits (Weight: 15%)

**Feature-to-Benefit Translation Check:**
- Bad: "AI-powered analytics dashboard"
- Good: "See exactly which campaigns drive revenue -- AI analyzes your data so you don't have to"

### Section 5: Objection Handling (Weight: 10%)

| Common Objection | How to Address |
|-----------------|----------------|
| "Too expensive" | ROI calculator, price comparison, guarantee |
| "Not sure it works" | Case studies, free trial, demo video |
| "Too complicated" | "Get started in 5 minutes", setup wizard |
| "What if I don't like it?" | Free trial, money-back guarantee, cancel anytime |
| "Is my data safe?" | Security badges, compliance logos |

### Section 6: Call-to-Action (Weight: 10%)

**CTA Copy Scoring:**
- Weak: "Submit", "Click Here", "Learn More"
- Medium: "Sign Up", "Get Started", "Download Now"
- Strong: "Start My Free Trial", "Get My Custom Report", "Claim Your Discount"

**Checklist:**
- [ ] CTA describes VALUE, not action
- [ ] Button is visually dominant
- [ ] Appears multiple times on long pages
- [ ] Secondary CTA for visitors not ready to commit
- [ ] Supporting microcopy (e.g., "No credit card required")
- [ ] First person ("Start MY trial")

### Section 7: Footer and Secondary Elements (Weight: 5%)

- [ ] Final CTA at bottom
- [ ] Contact info or support visible
- [ ] Privacy policy and terms linked
- [ ] Trust badges near final CTA
- [ ] No competing links that distract from conversion

## Step 3: Copy Scoring

Score on 5 dimensions (1-10 each):
1. **Clarity** -- Can visitor understand the offer in 5 seconds?
2. **Urgency** -- Is there a reason to act now?
3. **Specificity** -- Concrete numbers, timeframes, outcomes?
4. **Proof** -- Claims backed by evidence?
5. **Action Orientation** -- Does copy drive toward a next step?

Copy Score = average of all 5, multiplied by 10 (out of 100).

## Step 4: Form Optimization

| Element | Best Practice |
|---------|--------------|
| Field count | Every additional field reduces conversion ~7%. Lead capture: 3-5 max. |
| Labels | Inline or floating labels. No placeholder-only. |
| Button text | Match value proposition. |
| Error handling | Inline validation. Don't clear form on error. |
| Multi-step | Break long forms into steps with progress indicator. |

## Step 5: Mobile Audit

- [ ] CTA is thumb-reachable
- [ ] Text readable without zooming (16px minimum)
- [ ] Forms usable on mobile
- [ ] No horizontal scrolling
- [ ] Page loads under 3 seconds on 4G

## Step 6: Page Speed Impact

| Load Time | Conversion Impact |
|-----------|-------------------|
| 0-2 seconds | Baseline (optimal) |
| 2-3 seconds | -7% conversion rate |
| 3-5 seconds | -20% conversion rate |
| 5-8 seconds | -35% conversion rate |
| 8+ seconds | -50%+ conversion rate |

## Step 7: A/B Test Recommendations

Format each as a hypothesis:
"If we [CHANGE], then [METRIC] will [IMPROVE] because [REASON]."

**Tests to consider:**
1. Headline variations (benefit vs outcome focused)
2. CTA button color and text
3. Social proof placement
4. Form field count reduction
5. Hero image vs video
6. Long-form vs short-form page
7. Urgency elements
8. Price anchoring
9. Testimonial format
10. Adding live chat

## Key Principles

- Tie every recommendation to REVENUE IMPACT
- Prioritize by effort-to-impact ratio
- Be specific: not "improve headline" but "Change from X to Y because Z"
- Reference industry benchmarks
- Quick wins first, strategic changes later
