# Copywriting Analysis and Generation

Analyze existing website copy, score it against proven frameworks, and produce optimized alternatives with specific before/after examples.

## When to Use

- Auditing website or landing page copy
- Improving headlines, CTAs, and value propositions
- Generating copy alternatives for A/B testing
- Scoring copy quality against best practices

## Phase 1: Copy Discovery

### Fetch and Parse
Extract from the target page:
- Primary headline (H1) and subheadline
- Hero section copy
- All section headlines (H2, H3)
- Body copy paragraphs
- CTA button text (every instance)
- Meta title and meta description
- Social proof elements

### Detect Page Type

| Page Type | Primary Goal | Copy Priority |
|-----------|-------------|---------------|
| Homepage | Communicate value prop | Headline clarity, CTA hierarchy |
| Landing Page | Single conversion | Headline-CTA alignment, objection handling |
| Pricing Page | Drive plan selection | Plan naming, feature framing, FAQ |
| About Page | Build trust | Story, mission, credibility |
| Product Page | Demonstrate value | Feature-to-benefit translation |

### Voice Analysis

Score the existing voice on 5 dimensions (1-5 each):
- **Formality:** Casual to Formal
- **Emotion:** Neutral to Passionate
- **Complexity:** Simple to Technical
- **Humor:** Serious to Playful
- **Authority:** Peer to Expert

## Phase 2: Copy Scoring

### Headline Analysis (5-Second Test)
Would a new visitor understand what this company does within 5 seconds?

**Headline Scoring:**
- **Clarity (0-10):** Is meaning immediately obvious?
- **Specificity (0-10):** Concrete details? Numbers, outcomes, timeframes?
- **Relevance (0-10):** Speaks to target audience's primary pain?
- **Differentiation (0-10):** Sets business apart from competitors?
- **Emotion (0-10):** Triggers curiosity, desire, or recognition?

### Headline Formulas

**PAS (Problem-Agitate-Solve):**
"Stop [pain]. Start [desired outcome] -- with [product]."

**AIDA (Attention-Interest-Desire-Action):**
"[Bold claim] -- [specific outcome] in [timeframe]."

**Before-After-Bridge:**
"From [before state] to [after state] -- [product] makes it happen."

**4U Framework:**
Useful + Ultra-specific + Unique + Urgent

### Full Copy Scoring Rubric

| Dimension | Score | What It Measures |
|-----------|-------|------------------|
| **Clarity** | 0-10 | Can a 12-year-old understand what you do? |
| **Persuasion** | 0-10 | Does copy move reader toward action? |
| **Specificity** | 0-10 | Concrete numbers, outcomes, timeframes? |
| **Emotion** | 0-10 | Connects with pain, desires, aspirations? |
| **Action** | 0-10 | CTAs clear, compelling, strategically placed? |

**Total Copy Score: X/50** (multiply by 2 for 0-100 scale)

### Value Proposition Canvas

```
TARGET CUSTOMER: [Who specifically is this for?]
PROBLEM: [What painful problem do they have?]
SOLUTION: [How does this product solve it?]
UNIQUE MECHANISM: [What is the unique approach?]
KEY BENEFIT: [What is the #1 outcome?]
PROOF: [What evidence supports the claims?]
```

## Phase 3: Copy Generation

### CTA Optimization

**Best Practices:**
- Use first person: "Start My Free Trial" not "Start Your Free Trial"
- Include the value: "Get My Report" not "Submit"
- Reduce risk: "Try Free for 14 Days" not "Buy Now"
- Be specific: "Download the 2026 Marketing Guide" not "Download"
- Add urgency when appropriate: "Claim My Spot (12 Left)"

**CTA Placement Rules:**
- CTA above the fold (required)
- CTA after each major content section (recommended)
- Sticky/floating CTA on long pages (recommended)
- CTA repeated at bottom (required)

### Before/After Examples

For every recommendation, provide a concrete before/after:

```
BEFORE (Current):
  "We provide innovative solutions for businesses."

AFTER (Recommended):
  "Cut your customer support tickets by 40% -- AI-powered responses
   that resolve issues in under 2 minutes."

WHY: The "before" is vague and generic. The "after" is specific (40%),
outcome-driven, and includes a proof point (under 2 minutes).
```

Generate at least 5 before/after pairs covering:
1. Primary headline
2. Subheadline
3. Primary CTA
4. One body copy paragraph
5. Meta description

### Swipe File

Generate:
- 10 headline alternatives ranked by estimated effectiveness
- 5 subheadline alternatives
- 5 CTA button text alternatives
- 3 meta description alternatives
- 3 social proof framing alternatives

## Page-Specific Copy Structures

**Homepage:** Hero + Social proof bar + Problem + Solution + How it works + Features/Benefits + Testimonials + Final CTA

**Landing Page:** Headline + Subhead + Hero CTA + Problem + Solution + Benefits + Social proof + Objection handling + Final CTA

**Pricing Page:** Value-framing headline + 3 plans with recommended highlighted + Benefit-oriented features + FAQ addressing objections + Guarantee

**About Page:** Mission statement + Origin story + Values with examples + Team + Social proof + CTA

## Key Principles

- Always tie recommendations to revenue impact
- Every "fix" must include a concrete alternative, not just "improve the headline"
- Score honestly -- do not inflate scores
- Focus on changes that directly affect conversions
- Match generated copy to the brand's existing voice unless it is clearly ineffective
